\begin{abstract}
We sketch how to repurpose the OKVS-based, RLWE-instantiated Reverse-PMT (RPMT) of the
companion paper into a \emph{private encrypted data alignment} primitive: a receiver
$\Rcv$ holds a table $\{(\mathsf{ID}_i,\mathsf{payload}_i)\}$ and a sender $\Snd$ holds a
set of IDs; for each of $\Snd$'s IDs that matches one of $\Rcv$'s, $\Snd$ obtains an
\emph{encryption} of the corresponding payload, aligned to $\Snd$'s own order, so that
$\Snd$ can run a homomorphic analysis on the aligned payloads together with its own data.
The two non-trivial pieces are (i) neutralizing the OKVS ``garbage'' on non-matching IDs
without leaking the join pattern, via a secret-shared membership indicator and a single
ciphertext$\times$ciphertext mask, and (ii) carrying real-valued payloads in CKKS, which
turns the OKVS encoding problem from a finite-field full-rank question into a
\emph{numerical conditioning} question. We record the design decisions and the resulting
research questions.
\end{abstract}

\section{Setting and target functionality}

Let $\Rcv$ (receiver, data owner) hold a key--value table
$T=\{(\mathsf{ID}_i,\mathsf{payload}_i)\}_{i\in[n_x]}$ and let $\Snd$ (sender) hold a set of
identifiers $\{\mathsf{ID}^\Snd_j\}_{j\in[n_y]}$. We want a protocol that delivers to $\Snd$,
for each $j$, an \emph{encryption} of $\mathsf{payload}_i$ whenever
$\mathsf{ID}^\Snd_j=\mathsf{ID}_i$ for some $i$, and an encryption of (effectively) zero
otherwise; the result is aligned to $\Snd$'s own indexing. $\Snd$ then evaluates an
application-specific homomorphic function on these aligned ciphertexts and its own
plaintext data. Throughout, $\Rcv$ holds the RLWE secret key(s); all ciphertexts handed to
$\Snd$ are under $\Rcv$'s public key, so $\Snd$ computes but cannot decrypt. The final
analysis output is returned to $\Rcv$ (or kept secret-shared); this delivery and the choice
of analysis are treated as out of scope here.

\section{Recap of the base RPMT mechanism}

The base construction (companion paper) realizes
$F_{\mathrm{RPMT}}$: $b_j=\mathbf{1}(y_j\in X)$. Its mechanics that we reuse:
\begin{enumerate}[leftmargin=*]
  \item $\Rcv$ OKVS-encodes $\{(x_i',\IS)\}$ --- \emph{same} indicator value $\IS$ for
  every key --- into $p$, using the \emph{Random Spaced Band} (RSB) matrix (a fixed column
  permutation $\pi_N$ of a random-band matrix, aligning bands to the RLWE slot size $N$).
  \item $\Rcv$ RLWE-encrypts $p$ (block-packed, plus pre-rotated copies) and sends it.
  \item $\Snd$ homomorphically evaluates the OKVS decoding $M\cdot p$ as a
  (block-)diagonal matrix--vector product, using only slot-wise $\PlainMult/\CtxtAdd$.
  \emph{Sequencing} (Algorithm~3 of the paper) splits the collision-laden matrix into $L$
  layers $\bar M_t\in\Zp^{N\times m}$, each with $\le w+\zeta$ diagonals; the assignment of
  item $i$ to layer/slot $(\mu_i,\tau_i)$ depends only on the band start
  $a_i=H_1(k_i)$, the slot count $N$, and the span limit $\zeta$ --- \emph{not} on $w$.
  \item Each decoded slot is $\IS$ if $y_j\in X$ and a pseudorandom value otherwise. The
  base protocol re-randomizes, returns the $L$ ciphertexts to $\Rcv$, who tests equality
  with $\IS$.
\end{enumerate}
A key efficiency point: sequencing is an $n$-balls-into-$N$-bins process, so its batching
efficiency relies on the regime $N\ll n$ (well-filled slots). The base parameters fix
$N=2^{13}$ for all $n\in[2^{16},2^{22}]$.

\paragraph{Secret-shared PMT (Appendix~D).} Masking the decoded plaintext by $r_\mu$ before
$\Rcv$ decrypts, followed by a secret-shared equality test $F_{\mathrm{ss\text{-}peqt}}$,
yields XOR shares $b_{X,i}$ (held by $\Rcv$) and $b_{Y,i}$ (held by $\Snd$) with
$b_{X,i}\oplus b_{Y,i}=\mathbf 1(y_i\in X)$. We reuse this verbatim for the indicator.

\section{The alignment construction}

The construction runs \emph{two} OKVS instances on the same OPRF'd keys
$k_i=\OPRF(\mathsf{ID}_i)$: an \emph{indicator} OKVS over a finite field (all values $=\IS$,
exactly the base protocol, used only to obtain membership shares), and a \emph{payload}
OKVS whose value for $k_i$ is $\mathsf{payload}_i$, decoded homomorphically so that $\Snd$
holds a ciphertext whose slot for $y_j$ contains $\mathsf{payload}_i$ on a match and
pseudorandom garbage on a non-match. A generic payload no longer self-certifies membership
(garbage is indistinguishable from a real payload), which is exactly why the separate
indicator is needed: it provides a secret-shared membership bit with which the garbage is
masked off. The receiver-side encoding is Algorithm~\ref{alg:encode}; the sender-side
alignment-and-masking is Algorithm~\ref{alg:align}.

\begin{algorithm}[h]
\caption{$\Pi.\mathsf{Encode}$ --- receiver $\Rcv$, on table $\{(\mathsf{ID}_i,\mathsf{payload}_i)\}_{i\in[n_x]}$}
\label{alg:encode}
\begin{algorithmic}[1]
\State $k_i \gets \OPRF(\mathsf{ID}_i)$ for all $i$ \Comment{OPRF preprocessing (R is OPRF receiver)}
\State $\IS \gets$ random indicator tag
\State $p^{\mathrm{ind}} \gets \textsc{Okvs.Encode}_{\Zp}\big(\{(k_i,\IS)\}_i\big)$ \Comment{indicator OKVS, finite field}
\State $p^{\mathrm{pay}} \gets \textsc{Okvs.Encode}_{\mathbb R}\big(\{(k_i,\mathsf{payload}_i)\}_i\big)$ \Comment{payload OKVS, min-norm solve (Alg.~\ref{alg:encode-real})}
\State $C^{\mathrm{ind}} \gets \mathsf{BGV.Enc}(p^{\mathrm{ind}})$,\quad $C^{\mathrm{pay}} \gets \mathsf{CKKS.Enc}(p^{\mathrm{pay}})$ \Comment{block-packed}
\State \textbf{send} $C^{\mathrm{ind}}, C^{\mathrm{pay}}$ to $\Snd$ \Comment{(rotation/Galois keys for the spaced-band decode are sent in setup)}
\end{algorithmic}
\end{algorithm}

\begin{algorithm}[h]
\caption{$\Pi.\mathsf{Align}$ --- sender $\Snd$, on set $\{y_j\}_{j\in[n_y]}$ (and $\Rcv$'s share step)}
\label{alg:align}
\begin{algorithmic}[1]
\State $(c^{\mathrm{ind}}_\mu)_\mu \gets \textsc{HomDecode}(C^{\mathrm{ind}}, \{y_j\})$ \Comment{base-protocol indicator decode}
\State $(c^{\mathrm{pay}}_\mu)_\mu \gets \textsc{HomDecode}(C^{\mathrm{pay}}, \{y_j\})$ \Comment{CKKS spaced-band decode, Alg.~\ref{alg:homdecode}}
\State run ssPMT on $(c^{\mathrm{ind}}_\mu)_\mu$: $\Snd$ gets share $b_{Y,j}$, $\Rcv$ gets $b_{X,j}$, with $b_{X,j}\oplus b_{Y,j}=\mathbf 1(y_j\in X)$
\Statex \hrulefill\ \emph{garbage removal (deferred; see Section~\ref{sec:real} note)} \hrulefill
\State $\Rcv$ \textbf{sends} $\Enc_{\mathrm{CKKS}}(b_{X,j})$, packed into the payload slot layout
\State $\Snd$ sets $s_j \gets b_{Y,j}$ and forms $\Enc(b_j) \gets \PlainAdd\big(\PlainMult(\Enc(b_{X,j}),\,1-2s_j),\ s_j\big)$ \Comment{XOR by a \emph{plaintext} bit: no ctxt$\times$ctxt}
\State $\Snd$ computes $\Enc(\widetilde{\mathsf{payload}}_j) \gets \Enc(b_j)\odot c^{\mathrm{pay}}_{(\cdot)}[j]$ \Comment{the single ctxt$\times$ctxt mult; $1\!\cdot\!\mathsf{payload}$ or $0\!\cdot\!\text{garbage}$}
\State \Return aligned $\{\Enc(\widetilde{\mathsf{payload}}_j)\}_j$ \Comment{$\Snd$ runs its analysis; circuit-privacy/noise-flood before $\Rcv$ decrypts}
\end{algorithmic}
\end{algorithm}

Two facts about Algorithm~\ref{alg:align}. The XOR in line~5 uses $a\oplus b=a+b-2ab$ (not
$a+b-ab$, which is OR); since $\Snd$'s share $s_j$ is a \emph{plaintext} bit, line~5 is a
plaintext multiply-add, with \emph{no} ciphertext$\times$ciphertext product. Line~6 is the
\emph{only} ciphertext$\times$ciphertext multiplication --- it takes the protocol out of the
base paper's additive-only regime by one multiplicative level (plus relinearization), to be
budgeted in the modulus. The membership bit $b_j$ never appears in the clear to either party,
so the join pattern stays hidden from $\Snd$.

\begin{remark}[circuit privacy]
Since $\Snd$'s computation touches the ciphertext before $\Rcv$ decrypts the final analysis
output, noise-flooding / circuit privacy must be applied before that decryption, exactly as
the base protocol does on its returned ciphertexts.
\end{remark}

\subsection{Scheme split: indicator in BGV/BFV, payload in CKKS}
Real-world payloads are typically real numbers, so the payload OKVS is naturally carried in
CKKS while the indicator/equality logic stays in BGV/BFV. The re-encryption of $b_{X,i}$ into
CKKS is free: the share is plaintext to its holder, so no scheme-switching
primitive is required --- $\Rcv$ simply CKKS-encrypts a fresh $0/1$, and the plaintext-bit
XOR of Algorithm~\ref{alg:align} (line~5) runs in CKKS. The only semantic change from the BGV masking is that \emph{exact} zeroing becomes
\emph{approximate}: with $b_i=0\pm\delta$, a non-match leaves a residual $\delta\cdot
\text{garbage}$. Hence the central numerical requirement:
\[
  \norm{p}\le \frac{\norm{v}}{\sigmin(M)},\qquad
  |\text{garbage}|=|\langle\row(y'),p\rangle|\le \sqrt{w}\,\norm{p},\qquad
  \text{CKKS residual}\sim \delta\cdot|\text{garbage}|,
\]
so controlling $\sigmin(M)$ (equivalently $\norm{p}$) is what keeps the residual below the
analysis tolerance (Section~\ref{sec:real}).

\subsection{Slot geometry and the ``promote later'' parameter plan}
BGV/BFV with ring dimension $N$ has $N$ slots; CKKS with ring dimension $N'$ has $N'/2$
slots. To share one sequencing grid between the two decodes (so $\Rcv$ packs its membership
share once, with no cross-grid permutation), the slot counts must match. The clean way is to
choose the CKKS ring as $N'=2N$, which CKKS likely needs anyway for the masking depth and
payload precision. Then both have $N$ slots and the $(\mu_i,\tau_i)$ layout is identical
(it depends only on $H_1$, the slot count, and $\zeta$; $w$ and the band values may differ
between the two instances).

Two further points drive the parameterization:
\begin{itemize}[leftmargin=*]
  \item \textbf{Keep $N$ small during alignment.} Sequencing is $n$-balls-into-(slot count);
  large slot counts under-fill the bins and waste computation on near-empty ciphertexts. The
  alignment phase should run at modest $N$ (e.g.\ BFV $2^{13}$ / CKKS $2^{14}$), preserving
  $N\ll n$.
  \item \textbf{Bootstrap is not a dimension lift.} $N$ is fixed at key generation;
  bootstrapping refreshes the modulus/level at the \emph{same} $N$. If deep analysis is
  needed, one keeps alignment at small $N$ and, \emph{after} masking (alignment done),
  ring-switches the aligned payload \emph{up} to a large ring and bootstraps there
  (a sparse-slot, hence favorable, bootstrapping). The big-$N$ bootstrapping keys must be
  provisioned upfront, but the shallow alignment work stays cheap at small $N$. Ring
  switching \emph{down} (large$\to$small) is the genuinely useful direction for shrinking
  the tail; ring switching \emph{up} grants no fresh noise budget by itself --- only the
  subsequent bootstrap does.
\end{itemize}

\section{Real-field OKVS: conditioning instead of rank}\label{sec:real}

For the payload OKVS over $\mathbb{R}$ (CKKS) we use \emph{Gaussian} random bands: each
width-$w$ band has i.i.d.\ $\mathcal N(0,1)$ coefficients, placed at a random start. The
encoding solves $M p=v$ over $\mathbb{R}$, where $M$ is $n\times m$,
$m\approx(1+\varepsilon)n$.

\paragraph{Why Gaussian, not binary.}
The earlier motivation for $0/1$ bands was an ``addition-only'' homomorphic decode, but this
is illusory: the decode is $\sum_k \mathsf{PlainMult}(\rho_k(c),\,\mathrm{diag}_k)$, and a
$\mathsf{PlainMult}$ costs the same whether the plaintext diagonal is $0/1$ or real (it is
encoded into a dense plaintext polynomial and NTT-multiplied regardless); binary also gives
no encoding-time speedup off $\mathrm{GF}(2)$. Meanwhile full-entropy real coefficients
condition markedly better --- experimentally $\sigmin$ is $\sim\!1.7\times$ larger at equal
$w$, so Gaussian bands reach a target $\sigmin$ with $\sim\!1/3$ fewer diagonals --- and
admit clean random-matrix tools (continuous anti-concentration) for the analysis below.

\paragraph{Rank is automatic; conditioning is the whole question.}
For Gaussian bands full row rank holds almost surely (the only structural requirement is that
the sparsity pattern admits a row-to-column matching, which holds once $(\varepsilon,w)$
clear the RB-OKVS thresholds). So unlike the finite-field case there is no rank-failure event
to bound; CKKS instead needs $\sigmin(M)$ bounded \emph{away from $0$}, since a full-rank $M$
with tiny $\sigmin$ already yields an exploding $\norm{p}$. The operative target is a
\emph{smallest-singular-value tail bound}
\[
  \Pr[\sigmin(M)<t]\le 2^{-\lambda},
\]
which replaces the finite-field rank-failure bound and which sets $(\varepsilon,w)$; a proof
sketch is given in Section~\ref{sec:sigmin}. Redundancy $\varepsilon$ (extra columns) is the
primary knob --- it sets the spectral gap $\sigmin\gtrsim\sqrt m-\sqrt n$ --- with $w$
secondary, as the experiments confirm.

\paragraph{Efficient minimum-norm encoding.}
Among all valid encodings $\{p_0+\ker M\}$ the minimum-$\ell_2$-norm solution is
$p^\star=M^\top(MM^\top)^{-1}v$, and its size is governed precisely by $\sigmin$:
$\norm{p^\star}\le\norm{v}/\sigmin(M)$. Crucially this is cheap to compute for band
matrices: ordering rows by band start makes $MM^\top$ \emph{banded} (bandwidth $O(w)$), so
$MM^\top y=v$ is solved by banded Cholesky in $O(nw^2)$ and $p^\star=M^\top y$ in $O(nw)$ ---
comparable to the encoding itself, and far smaller in norm than the naive triangular
particular solution (free variables set to $0$), whose norm is governed by the worse-
conditioned pivot submatrix. (Normal equations square the condition number; a banded QR of
$M^\top$ is the stable fallback.) The encoding and the spaced-band homomorphic decode are
Algorithms~\ref{alg:encode-real} and~\ref{alg:homdecode}.

\begin{algorithm}[h]
\caption{$\textsc{Okvs.Encode}_{\mathbb R}$ --- minimum-norm real-field OKVS encoding}
\label{alg:encode-real}
\begin{algorithmic}[1]
\Require keys $\{k_i\}_{i\in[n]}$, values $v\in\mathbb R^n$, width $w$, expansion $\varepsilon$ ($m=\lceil(1+\varepsilon)n\rceil$)
\State for each $i$: start $a_i\gets H_1(k_i)\in[0,m{-}w]$; in-band coeffs $(M_{i,a_i+j})_{j<w}\gets H_2(k_i)\sim\mathcal N(0,1)$
\State sort rows by $a_i$ and form the banded SPD Gram matrix $G=MM^\top$ (bandwidth $O(w)$)
\State $G=LL^\top$ \Comment{banded Cholesky, $O(nw^2)$; SPD $\Rightarrow$ pivot-free, stable}
\State solve $LL^\top y=v$ by two banded triangular solves \Comment{$O(nw)$}
\State \Return $p^\star=M^\top y$ \Comment{min-norm solution; $\norm{p^\star}\le\norm{v}/\sigmin(M)$}
\end{algorithmic}
\end{algorithm}

\begin{algorithm}[h]
\caption{$\textsc{HomDecode}_{\mathrm{CKKS}}$ --- spaced-band decode, rotation-key variant}
\label{alg:homdecode}
\begin{algorithmic}[1]
\Require CKKS ciphertexts $(c_0,\dots,c_{b-1})$ of $p=(p_0\Vert\cdots\Vert p_{b-1})$, $p_i\in\mathbb R^{N}$ ($N$ slots, $b=m/N$); query keys $\{y_j\}$; Galois (rotation) keys
\State build the RSB query matrix $M'$ from $\{y_j\}$ ($N$-spaced bands); diagonalize and sequence into layers $\{\bar M_t\}$, each with offset set $D_t\subseteq\{0,\dots,w{+}\zeta{-}1\}$
\For{each distinct offset $k\in\bigcup_t D_t$}
  \State $\rho_k \gets \textsc{HomRotate}(c_{\bullet},\,k)$ \Comment{homomorphic rotation via Galois key, computed \emph{once}, reused across layers}
\EndFor
\For{each layer $t$}
  \State $c^{*}_t \gets \sum_{k\in D_t}\PlainMult(\rho_k,\ \mathrm{diag}_{t,k})$ \Comment{$w{+}\zeta$ slot-wise plain-mult/adds}
\EndFor
\State \Return $\{c^{*}_t\}_t$ \Comment{slot of $y_j$ holds $\langle\row(y_j),p\rangle=$ payload (match) or garbage (non-match)}
\end{algorithmic}
\end{algorithm}

\begin{remark}[implementation status]\label{rmk:garbage}
The accompanying CKKS implementation realizes Algorithms~\ref{alg:encode-real}
and~\ref{alg:homdecode}: the encode/decode round-trip that delivers
$\langle\row(y_j),p\rangle$ in the encrypted state, including the multi-block $N$-spacing
($m=bN$) with Galois-key homomorphic rotations for the super-diagonals (rotation by $r$
computed once per block, reused across layers), verified correct for all queries at scale
$2^{40}$. Deferred: the \emph{clustered} sequencing of Algorithm~3 of the base paper (the
current sequencing is naive, one query per slot per layer), and masking off the non-match
garbage (lines~4--6 of Algorithm~\ref{alg:align}, needing the indicator OKVS and the
ciphertext$\times$ciphertext multiply).
\end{remark}

\section{The $\sigmin$ tail bound}\label{sec:sigmin}

\paragraph{Model.} $M\in\mathbb R^{n\times m}$, $m=\lceil(1+\varepsilon)n\rceil$; row $i$ is
supported on $[a_i,a_i+w)$ with $a_i\sim U[0,m-w]$ i.i.d., and the $w$ in-band entries are
i.i.d.\ $\mathcal N(0,1)$. We write $r_1,\dots,r_n$ for the rows.

\begin{theorem}[Conditioning of Gaussian band matrices]\label{thm:sigmin}
Fix $\varepsilon_0\in(0,1]$. There are constants $c_1,c_2,c_3>0$ depending only on
$\varepsilon_0$ such that for every $n$, every $\varepsilon\ge\varepsilon_0$, and every width
$c_1\log n\le w\le n$, the model above satisfies
\[
   \Pr\big[\,\sigmin(M)<c_2\,\big]\;\le\;e^{-c_3\,\min(\varepsilon n,\,w)} .
\]
More precisely there is a threshold $t(\varepsilon,w)$, non-decreasing in both arguments and
bounded below by $c_2>0$, with $\Pr[\sigmin(M)<t(\varepsilon,w)]\le e^{-c_3\min(\varepsilon n,w)}$.
In particular $\sigmin(M)$ is bounded away from $0$ by a constant \emph{independent of $n$}
with probability $1-e^{-\Omega(\min(\varepsilon n,w))}$: the conditioning does not degrade with
set size.
\end{theorem}

The bound is governed by $\min(\varepsilon n, w)$. For the set sizes of interest $\varepsilon
n\gg w$, so the binding term is $w$: $\log\Pr[\text{ill-conditioned}]\approx -c_3 w$ is
\emph{linear in $w$}, and a target failure $2^{-\lambda}$ needs $w=\Omega(\lambda)$ --- the
prediction tested empirically in the companion experiment. The next corollary turns the
$\sigmin$ bound into the two quantities the construction actually needs: a bounded encoding
and a bounded encode/decode precision loss.

\begin{corollary}[Bounded encoding and round-trip precision]\label{cor:precision}
Under Theorem~\ref{thm:sigmin}, except with probability $e^{-c_3\min(\varepsilon n,w)}$:
\begin{enumerate}[label=(\roman*),leftmargin=*,topsep=2pt]
  \item the min-norm encoding obeys $\norm{p^\star}\le \norm{v}/c_2$, a bound independent of
  $n$ (so the CKKS garbage $|\langle\row(y'),p^\star\rangle|\le\sqrt w\,\norm{p^\star}$ is
  likewise $O(\sqrt w\,\norm v)$);
  \item if $\Snd$ decodes from the CKKS ciphertext of $p^\star$ held with error
  $\tilde p=p^\star+e$, $\norm e_2\le\eta$, then every matching query $y_j=x_i$ recovers
  \[
     v'_j=\langle\row(y_j),\tilde p\rangle = v_i + \langle\row(y_j),e\rangle,\qquad
     |v'_j-v_i|\le \norm{\row(y_j)}_2\,\eta=\Theta(\sqrt w)\,\eta .
  \]
\end{enumerate}
Hence preserving $b$ bits of payload precision through the encode/decode round-trip costs
$\approx \tfrac12\log_2 w+\log_2(1/\sigmin)=O\!\big(\log(\sqrt w/\sigmin)\big)$ extra bits of
CKKS scale --- the $1/\sigmin$ factor is the encoding's norm inflation, the $\sqrt w$ the
band-sum amplification.
\end{corollary}

\begin{proof}[Proof of Corollary~\ref{cor:precision}]
(i) is $\norm{p^\star}\le\norm v/\sigmin(M)\le\norm v/c_2$ on the high-probability event of
Theorem~\ref{thm:sigmin}, and $|\langle\row(y'),p^\star\rangle|\le\norm{\row(y')}\norm{p^\star}$
with $\norm{\row(y')}=\Theta(\sqrt w)$. (ii): for a matching key $\row(y_j)=\row(x_i)$, so
$\langle\row(y_j),p^\star\rangle=v_i$ by OKVS correctness; the error is
$\langle\row(y_j),e\rangle$, bounded by Cauchy--Schwarz. The bit count follows by setting
$\sqrt w\,\eta$ below the payload LSB and noting $\eta$ scales with $\norm{p^\star}$, which
carries the $1/\sigmin$ factor.
\end{proof}

\paragraph{Proof sketch (full proof in Appendix~\ref{app:proof}).}
Write $\sigmin(M)=\min_{u\in S^{n-1}}\norm{M^\top u}_2$, $M^\top u=\sum_i u_i r_i$. The band
structure makes the $m$ coordinates of $M^\top u$ \emph{independent} Gaussians of total
variance $w$ (distinct columns use disjoint entries: $(M^\top u)_j\sim\mathcal N(0,s_j^2(u))$,
$s_j^2(u)=\sum_{i:\,j\in[a_i,a_i+w)}u_i^2$, $\sum_j s_j^2=w$). Tensorizing the per-coordinate
Gaussian anti-concentration gives a single-direction small ball $(C\rho)^{\kappa(u)}$, with
$\kappa(u)=\#\{j:s_j^2(u)>0\}$ the number of \emph{active columns} the direction touches. One
then splits $S^{n-1}$: \emph{incompressible} directions spread over $\kappa=\Theta(\varepsilon
n)$ columns (beating the $e^{O(n)}$ spherical net), \emph{compressible} ($\approx k$-sparse)
ones still touch $\kappa=\Omega(w)$ columns (beating the small $k$-sparse net once
$w=\Omega(\log n)$); random starts make the pattern well-spread except with probability
$e^{-\Omega(w)}$ (balls-into-bins). Together these give the $\min(\varepsilon n,w)$ exponent.
Thus $\varepsilon$ sets the conditioning scale (spectral gap) and the incompressible exponent,
$w$ the compressible exponent --- matching the experiment, where $\sigmin$ rises with both and
collapses as $\varepsilon\to0$. Gaussian coefficients make the tensorized step the clean
Gaussian small ball; binary would need a Carbery--Wright / Berry--Esseen replacement.

\section{Open problems / next steps}

\begin{problem}[sharpen the $\sigmin$ threshold]
Theorem~\ref{thm:sigmin} (proof in Appendix~\ref{app:proof}) gives the qualitative tail
$\Pr[\sigmin<c_2]\le e^{-c_3\min(\varepsilon n,w)}$. What remains is the \emph{tight} threshold
$t(\varepsilon,w)$ --- an explicit $(\varepsilon,w)\mapsto t$ matching the measured surface
(and hence the smallest $w$ for a target $\sigmin$). The compressible-net balancing
(Appendix Step~B.3) and the optimal constant in the spectral gap are the parts needing care.
\end{problem}

\begin{problem}[empirical conditioning]
Measure, for Gaussian band matrices over $\mathbb{R}$: the amplification
$\norm{p}/\norm{v}$ for naive-triangular vs.\ banded-min-norm; $\sigmin(M)$ and its
dependence on $(w,\varepsilon)$; and the resulting garbage magnitude
$|\langle\row(y'),p\rangle|$ on non-matches. (Addressed by the accompanying experiment, which
also confirms Gaussian bands beat binary at equal $w$.)
\end{problem}

\begin{problem}[2-row decode]
The $2\times(N/2)$ BGV hypercube admits, with $N/2$-based sequencing and rotation-during-
compute, two layer products per row-wise rotation --- halving plain-mults for the indicator
decode and resolving the open remark of \S4.3 of the base paper. The packing (offset
alignment of paired layers; replication vs.\ windowed layout) and its
communication/computation trade-off remain to be fixed.
\end{problem}

\appendix
\section{Full proof of Theorem~\ref{thm:sigmin}}\label{app:proof}

Throughout, $c,C,c_i,C_i$ denote positive constants depending only on $\varepsilon_0$, and we
condition on the starts $\{a_i\}$ until Step~B.4. Recall $\sigmin(M)=\min_{u\in S^{n-1}}\norm{M^\top u}_2$ and $M^\top u=\sum_i u_i r_i$.

\paragraph{B.1 Independent-coordinate representation.}
For $u\in S^{n-1}$ the coordinates of $M^\top u$ are
$(M^\top u)_j=\sum_{i:\,j\in[a_i,a_i+w)}u_i M_{ij}$. Entries $\{M_{ij}\}$ with distinct $j$ are
disjoint sets of i.i.d.\ $\mathcal N(0,1)$ variables, so $\{(M^\top u)_j\}_{j\in[m]}$ are
independent with $(M^\top u)_j\sim\mathcal N(0,s_j^2(u))$,
$s_j^2(u)=\sum_{i:\,j\in[a_i,a_i+w)}u_i^2$, and $\sum_{j}s_j^2(u)=w\norm u_2^2=w$. Let
$\kappa(u)=\#\{j:s_j^2(u)>0\}=\big|\bigcup_{i\in\mathrm{supp}(u)}[a_i,a_i+w)\big|$ be the
number of active columns.

\begin{lemma}[tensorized small ball; Rudelson--Vershynin]\label{lem:tensor}
There are constants $C_0,p_0<1$ such that for every fixed $u$ and every $\rho\le 1$,
\[
  \Pr\big[\norm{M^\top u}_2 \le \rho\sqrt w\big]\;\le\;(C_0\rho)^{\,\kappa(u)/2}.
\]
\end{lemma}
\begin{proof}[Proof]
Each Gaussian coordinate satisfies $\Pr[|(M^\top u)_j|\le \rho\, s_j(u)]\le \sqrt{2/\pi}\,\rho$.
Restricting to the $\kappa(u)$ active coordinates and applying the tensorization lemma of
Rudelson--Vershynin (independent coordinates, uniform single-coordinate small ball) gives
$\Pr[\sum_{j}(M^\top u)_j^2\le \rho^2\sum_j s_j^2(u)]\le (C_0\rho)^{\kappa(u)/2}$; substitute
$\sum_j s_j^2(u)=w$.
\end{proof}

\paragraph{B.2 Good sparsity pattern.}
Call the pattern $\{a_i\}$ \emph{good} if (a) every column lies in $O(w/(1+\varepsilon))$ bands
and (b) every $k\le n$ rows have band-union of size $\ge \tfrac12\min(kw,\,c\varepsilon n)$.
Since $a_i\sim U[0,m-w]$ i.i.d.\ with $m=(1+\varepsilon)n$, (a) is a balls-into-bins bound and
(b) follows because $k$ width-$w$ windows placed uniformly in $[0,m]$ cover
$\Omega(\min(kw,m))$ points in expectation with Chernoff concentration; both fail with
probability $\le e^{-c_4 w}$. Assume the pattern is good (charged to the final budget).

\paragraph{B.3 Net argument on incompressible and compressible vectors.}
Fix $\delta,\theta\in(0,1)$ constants (chosen below) and split $S^{n-1}$ into
$\mathrm{Comp}(\delta n,\theta)$ (vectors within Euclidean distance $\theta$ of some
$\delta n$-sparse unit vector) and its complement $\mathrm{Incomp}$.

\emph{Incompressible.} By the Rudelson--Vershynin structure lemma, every
$u\in\mathrm{Incomp}$ has $\ge c_5 n$ coordinates with $|u_i|\ge c_6/\sqrt n$; these spread
over, by goodness~(b), $\kappa(u)\ge \tfrac12\min(c_5 n\cdot w,\,c\varepsilon n)=\Omega(\varepsilon n)$
active columns. Lemma~\ref{lem:tensor} gives, for each fixed $u$,
$\Pr[\norm{M^\top u}\le\rho\sqrt w]\le (C_0\rho)^{\Omega(\varepsilon n)}$. Take a
$\tfrac{\rho}{2\sigma_{\max}}$-net $\mathcal N$ of $\mathrm{Incomp}$ with
$|\mathcal N|\le (C/\rho)^{\,n}$ (here $\sigma_{\max}(M)\le C_7(\sqrt w+\sqrt m)$ w.h.p.\ by
the Bai--Yin/$\sigma_{\max}$ bound, charged to the budget; $\norm{M^\top\cdot}$ is
$\sigma_{\max}$-Lipschitz). Choosing $\rho=\rho_0$ a small constant makes
$(C_0\rho_0)^{\Omega(\varepsilon n)}|\mathcal N|\le e^{-c_8\varepsilon n}$, and the Lipschitz
net-to-sphere transfer yields $\min_{u\in\mathrm{Incomp}}\norm{M^\top u}\ge \tfrac12\rho_0\sqrt w$
except with probability $e^{-c_8\varepsilon n}$.

\emph{Compressible.} It suffices (by the distance characterization of $\sigmin$) to bound
$\min_{u\in\mathrm{Sparse}(\delta n)}\norm{M^\top u}$. A $\delta n$-sparse direction activates,
by goodness~(b), $\kappa(u)\ge \tfrac12\min(\delta n\, w, c\varepsilon n)=\Omega(w)$ columns
(for $\delta$ small enough that $\delta n w\le c\varepsilon n$, i.e.\ $\delta\le c\varepsilon/w$,
giving $\kappa=\Omega(\delta n w)$; we instead take $k=\Theta(\varepsilon n/w)$ so
$\kappa=\Omega(\varepsilon n)$ when that is $\le w$ and $\Omega(w)$ otherwise). A
$\rho$-net of $\mathrm{Sparse}(k)\cap S^{n-1}$ has $\binom{n}{k}(C/\rho)^k\le
e^{O(k\log(n/k))}$ points. With $\kappa=\Omega(w)$ active columns per point,
Lemma~\ref{lem:tensor} gives per-point probability $(C_0\rho)^{\Omega(w)}$; the union is
$\le e^{O(k\log(n/k))-\Omega(w)}\le e^{-\Omega(w)}$ once $w\ge c_1\log n$ and $k\log(n/k)\le
\tfrac12\cdot\Omega(w)$, which holds for $k=O(w/\log n)$. Hence
$\min_{u\in\mathrm{Comp}}\norm{M^\top u}\ge c_9\sqrt w$ except with probability $e^{-c_{10}w}$.

\paragraph{B.4 Assembly.}
On the good-pattern event (probability $1-e^{-c_4 w}$) and the two net events
(probabilities $1-e^{-c_8\varepsilon n}$, $1-e^{-c_{10}w}$), every $u\in S^{n-1}$ has
$\norm{M^\top u}\ge c_{11}\sqrt w$, i.e.\ $\sigmin(M)\ge c_{11}\sqrt w\ge c_2$. A union bound
over the three failure events gives $\Pr[\sigmin(M)<c_2]\le e^{-c_3\min(\varepsilon n,w)}$,
proving Theorem~\ref{thm:sigmin}. The threshold can be taken $t(\varepsilon,w)=c_{11}\sqrt w$
up to the $\varepsilon$-dependence absorbed in the incompressible scale; pinning the sharp
$t(\varepsilon,w)$ (in particular the $\varepsilon\to0$ collapse, where the incompressible
exponent $\varepsilon n$ and the spectral gap both vanish) is the content of Open
Problem~1. \qed
